Compilar:
-Para compilar, escreva make remove, depois make, e rode primeiro o ./server (o server) em um terminal e o ./client em outro terminal
-O teclado será reconhecido no segundo terminal, que roda o client, e será exibido no primeiro terminal (o server).

Jogo:
-t para atirar;
-w para andar para cima;
-s para andar para baixo;
-q para sair do jogo.
